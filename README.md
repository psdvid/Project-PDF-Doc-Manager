🚀 Overview
The PDF Document Manager is a web-based application designed to simplify how users interact with PDF files. It allows users to upload, read, and search PDFs effortlessly — while also providing analytics to track reading behavior.

✨ Key Features
Upload & Read PDFs  
Users can upload PDF documents and navigate through pages using intuitive Next and Previous controls.

Smart Search  
Search PDFs by tags or upload date entered during the upload process for quick document retrieval.

User Analytics  
Gain insights into user engagement — track how many times readers click Next or Previous while reading.

Responsive Interface  
Built with a clean and user-friendly design for smooth document interaction.
